# 新西兰15天南北岛精华行程

这份15天的行程旨在最大化您的新西兰体验，快速领略北岛的魔幻风情后，将重点放在南岛，进行一次经典的逆时针大环线，涵盖冰川、峡湾、雪山和星空。

---

### **第一部分：北岛魔幻之旅 (3天)**

*   **第1天：抵达奥克兰，初见千帆之都**
    - 抵达奥克兰(AKL)，办理租车。下午登上天空塔，俯瞰城市和海港。

*   **第2天：霍比特村 & 罗托鲁瓦地热**
    - 上午：驱车前往玛塔玛塔，参加**霍比特村 (Hobbiton)** 之旅，沉浸在电影场景中。
    - 下午：抵达地热之城**罗托鲁瓦 (Rotorua)**，参观蒂普亚 (Te Puia) 的间歇泉和毛利文化村。

*   **第3天：北岛探索 & 准备南下**
    - 上午：参观怀奥塔普地热世界 (Wai-O-Tapu)，欣赏多彩的地热池。
    - 下午：在罗托鲁瓦湖边放松，或准备次日的飞行。

---

### **第二部分：南岛大环线 (12天)**

*   **第4天：飞往基督城，开启南岛**
    - 从罗托鲁瓦(ROT)飞往基督城(CHC)。提取新租车后，游览这座重生的花园城市。

*   **第5天：穿越亚瑟通道，抵达西海岸** (约250km)
    - 驾车穿越壮观的**亚瑟通道国家公园 (Arthur's Pass)**，这是南阿尔卑斯山脉中最美的通道之一。
    - 傍晚抵达西海岸小镇霍基蒂卡 (Hokitika)，可以去海滩寻找漂亮的石头，观赏日落。

*   **第6天：冰川探险** (约140km)
    - 南下至**弗朗茨·约瑟夫冰川 (Franz Josef Glacier)** 或 **福克斯冰川 (Fox Glacier)**。
    - 下午：进行冰川谷地徒步。如果预算和天气允许，强烈推荐参加直升机冰川徒步 (Heli-hike)，这是终极的冰川体验。

*   **第7天：西海岸风光，前往瓦纳卡** (约280km)
    - 上午：可去马瑟森湖 (Lake Matheson) 拍摄库克山的完美倒影。
    - 之后，沿着美丽的西海岸公路，穿越哈斯特通道 (Haast Pass)，沿途欣赏瀑布和蓝池 (Blue Pools)。
    - 傍晚抵达宁静的湖畔小镇**瓦纳卡 (Wanaka)**。

*   **第8天：瓦纳卡休整，奔赴皇后镇** (约70km)
    - 上午：在瓦纳卡湖边寻找著名的“孤独的树”，或者挑战罗伊峰步道 (Roys Peak Track) 的一部分，俯瞰壮丽湖景。
    - 下午：驱车翻越皇冠山脉公路，抵达“冒险之都”**皇后镇 (Queenstown)**。

*   **第9-10天：皇后镇的激情与宁静**
    - **第9天 (激情)**: 体验一项极限运动（蹦极/峡谷秋千/喷射快艇），乘坐天空缆车上山顶玩Luge滑板车。
    - **第10天 (宁静)**: 驱车前往风景如画的**格林诺奇 (Glenorchy)**，在“魔戒”小镇徒步，或乘坐TSS蒸汽船游览瓦卡蒂普湖。

*   **第11天：米尔福德峡湾一日游**
    - 参加从皇后镇出发的巴士-游船一日游，前往被誉为“世界第八大奇迹”的**米尔福德峡湾 (Milford Sound)**。在峡湾中，您将看到标志性的教冠峰、壮观的瀑布和可爱的野生动物。

*   **第12天：前往库克山** (约270km)
    - 从皇后镇出发，经过林迪斯山口 (Lindis Pass)，抵达**奥拉基/库克山国家公园**。
    - 下午可以进行一些短途徒步，适应环境，准备次日的重头戏。

*   **第13天：徒步库克山，夜宿蒂卡普** (约110km)
    - 上午：完成南岛最经典的**胡克山谷步道 (Hooker Valley Track)**，欣赏冰川、雪山和浮冰湖的壮丽景色。
    - 下午：驱车前往**蒂卡普湖 (Lake Tekapo)**，入住酒店。

*   **第14天：星空下的宁静**
    - 在蒂卡普湖边的好牧羊人教堂旁漫步，享受奶蓝色的湖水。
    - 晚上：在世界级的黑暗天空保护区参加约翰山天文台观星团，为新西兰之旅画上璀璨的句号。

*   **第15天：返回基督城，准备离开** (约230km)
    - 上午从蒂卡普出发，约3小时车程返回基督城机场(CHC)，还车并搭乘返程航班。
