# 环球旅行计划 - 欧洲篇

* github 计划最新详情：https://github.com/originrock/World_Travel

## 旅行总览

*   **总时长**: 分四阶段，每年约3个月，共计约12个月
*   **预计出发时间**: 2029年 - 2033年，每年10月至次年2月期间
*   **主要目的地**: 欧洲大陆各国
*   **旅行主题**: 历史文化探索、城市风光、美食之旅、自然景观

## 预算与准备

*   **预算**: 欧洲的消费水平差异较大，预算需分国家进行详细规划。整体预算将高于大洋洲，尤其是在瑞士和北欧国家。后续将为每个国家创建详细的预算计划。
*   **签证**: 需提前研究并申请申根签证，并了解非申根区国家（如克罗地亚的部分时期、英国等）的签证要求。
*   **交通**: 欧洲内部交通便利，将以铁路为主，辅以廉价航空和局部地区的自驾。
*   **住宿**: 以Airbnb和Booking.com上的公寓及酒店为主，体验当地生活。

**重要提示**:
*   以上为初步设想，具体行程将根据实际情况（如签证、季节、预算）进行调整。
*   每个国家的详细行程（交通、住宿、景点等）将分别制定。
*   冬季（10月-2月）是欧洲的淡季，游客较少，但白昼时间短，天气寒冷，需做好相应准备。北欧地区是体验冬季项目和极光的最佳时间。
