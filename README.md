# 环球旅行计划

这是一个用于规划和记录我的环球旅行的 GitHub 项目。

## 项目结构

本仓库的目的是帮助我（以及可能对环球旅行感兴趣的你！）有条不紊地规划每一个细节，包括：

-   **`README.md`**: 项目总览和说明。
-   **`global_plan/`**: 存放整个旅行的宏观计划和通用准备清单。
    -   `0_trip_overview.md`: 整体路线图、时间规划、总预算。
    -   `1_pre_trip_checklist.md`: 行前准备总清单（护照、保险、疫苗等）。
    -   `2_global_visas.md`: 全球签证跟踪和管理。
-   **`destinations/`**: 存放每个具体目的地的详细旅行计划。
    -   每个目的地（例如：`Japan/`，`Italy/`）都有一个独立的文件夹。
    -   每个目的地文件夹内包含以下详细文件：
        -   `README.md`: 该目的地总览。
        -   `1_itinerary.md`: 详细行程安排。
        -   `2_transportation.md`: 往返交通及当地交通方案。
        -   `3_attractions.md`: 景点与特色旅游项目。
        -   `4_lodging_food.md`: 衣食住行（住宿、餐饮）。
        -   `5_budget.md`: 该目的地预算明细。
        -   `6_visa_legal.md`: 签证与当地法规。
        -   `7_driving_rental.md`: 驾照换考与车辆租赁。
        -   `8_notes.md`: 旅行注意事项。

## 如何使用

您可以克隆此仓库，根据您的旅行计划修改和填充这些文件。每个 `.md` 文件都旨在记录相应主题的详细信息和决策过程。

## 一、大洋洲篇

### 旅行总览
-   **总时长**: 约3个月 (90天)
-   **预计出发时间**: 2026年2月初
-   **主要目的地**: 澳大利亚 (75天，包含悉尼长居与环澳自驾)、新西兰 (15天，南北岛精华游)。
-   **核心体验**: 城市探索、史诗级公路自驾、自然奇观、文化体验。

### 整体路线与时间线概览
-   **2026年2月1日 - 3月17日**: 澳大利亚悉尼长居与准备 (45天)。
-   **2026年3月18日 - 4月16日**: 环澳大利亚自驾 (30天)，探索东南部、红土中心及昆士兰海岸。
-   **2026年4月17日 - 5月1日**: 新西兰南北岛精华游 (15天)，体验纯净自然美景。

## 二、北美篇

### 旅行总览
-   **总时长**: 约90天 (3个月)
-   **预计出发时间**: 2027年10月1日
-   **预计结束时间**: 2027年12月29日
-   **主要目的地**: 美国 (60天：西海岸、南部、东南部、东海岸)、加拿大 (30天：东部、西部)。
-   **核心体验**: 从美国西海岸开始，横跨大陆向东，感受不同地域的文化与自然风光；随后北上探索加拿大东部城市，再飞往西部领略其壮丽景色。

### 整体路线与时间线概览
-   **2027年10月1日 - 11月29日**: 美国之旅 (60天)，涵盖旧金山、洛杉矶、休斯顿、佛罗里达、亚特兰大、华盛顿特区、纽约等。
-   **2027年11月30日 - 12月29日**: 加拿大之旅 (30天)，涵盖蒙特利尔、多伦多、温哥华。

## 三、东北亚篇

### 旅行总览
-   **总时长**: 约60天 (2个月)
-   **预计出发时间**: 2028年4月1日
-   **预计结束时间**: 2028年5月30日
-   **主要目的地**: 日本 (30天：东京、京都、大阪、北海道)、韩国 (30天：首尔、釜山、济州岛)。
-   **核心体验**: 探索日本的现代与传统文化，从繁华都市到宁静古都，再体验韩国的流行文化、美食和海岛风光。

### 整体路线与时间线概览
-   **2028年4月1日 - 4月30日**: 日本之旅 (30天)，涵盖东京、京都、大阪、北海道等。
-   **2028年5月1日 - 5月30日**: 韩国之旅 (30天)，涵盖首尔、釜山、济州岛等。
