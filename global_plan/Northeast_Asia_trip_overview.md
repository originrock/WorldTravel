# 东北亚旅行总览 - 60天行程细化

## 旅行总览

-   **总时长**: 约60天 (2个月)
-   **预计出发时间**: 2028年4月1日
-   **预计结束时间**: 2028年5月30日
-   **主要目的地**: 日本 (30天：东京、京都、大阪、北海道)、韩国 (30天：首尔、釜山、济州岛)
-   **核心体验**: 探索日本的现代与传统文化，从繁华都市到宁静古都，再体验韩国的流行文化、美食和海岛风光。
