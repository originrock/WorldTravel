# 韩国的通信与网络

本文件旨在帮助您在韩国旅行期间选择最合适的手机和网络连接方案。韩国以其世界领先的网速和网络覆盖而闻名。

## 主要方案对比

| 方案类型 | 优点 | 缺点 | 推荐人群 |
| :--- | :--- | :--- | :--- |
| **eSIM** | - **极其方便**：出发前在线购买，通过二维码即时激活，无需换卡。<br>- **无需排队**：落地即可上网，节省在机场办理的时间。<br>- **保留原卡**：可同时接收原手机号的短信和电话。 | - 手机必须支持eSIM功能。<br>- 价格可能略高于物理SIM卡。 | 追求便捷、手机支持eSIM的个人游客。 |
| **物理SIM卡** | - **兼容性好**：适用于几乎所有无锁手机。<br>- **套餐灵活**：提供纯数据或包含本地通话/短信的套餐。<br>- **价格实惠**：选择多样，性价比高。 | - 需要更换手机卡，可能不便。<br>- 需在机场或市区柜台办理，可能排队。 | 对手机兼容性有要求、或需要本地通话功能的游客。 |
| **便携Wi-Fi (Wi-Fi Egg)** | - **一机多用**：可同时连接多个设备（手机、笔记本电脑等）。<br>- **适合团体**：多人共享，成本效益高。<br>- **无限流量**：通常提供无限4G LTE数据。 | - 需要额外携带一个设备并为其充电。<br>- 领取和归还设备略有不便。<br>- 需要支付押金。 | 携带多个设备、或与家人朋友一同出行的团体。 |

## 方案详解

### 1. eSIM (数字SIM卡)

-   **如何购买**:
    -   在旅行前通过 **Holafly**, **Airalo**, **Klook**, **Trazy** 等在线平台购买。
    -   韩国三大运营商 (SK Telecom, KT, LG U+) 也提供面向游客的eSIM产品。
-   **如何激活**:
    -   购买后会收到一个二维码，在手机的网络设置中扫描即可添加新的蜂窝网络计划。
-   **注意事项**:
    -   **必须确保您的手机是无锁的，并支持eSIM功能。** iPhone 11及之后的大部分型号、Google Pixel、三星部分旗舰机型都支持。

### 2. 物理SIM卡

-   **在哪里购买**:
    -   **机场**: 仁川(ICN)、金浦(GMP)、金海(PUS)等国际机场的运营商柜台或便利店。
    -   **市区**: 运营商门店或指定的便利店。
-   **需要什么**:
    -   通常需要出示**护照**进行实名登记。
-   **套餐类型**:
    -   **纯数据卡**: 只提供上网功能。
    -   **数据+通话卡**: 提供上网、本地通话和短信功能。

### 3. 便携Wi-Fi (Pocket Wi-Fi / Wi-Fi Egg)

-   **如何预订**:
    -   强烈建议**提前在线预订**，以确保设备可用并享受折扣。可通过Klook, Trazy, 或运营商官网预订。
-   **领取与归还**:
    -   通常在各大国际机场的指定柜台领取和归还。
    -   领取时需要出示预订凭证和**信用卡**以支付押金。

## 免费 Wi-Fi

韩国的公共Wi-Fi覆盖非常广泛，质量也很高。

-   **覆盖地点**:
    -   **公共交通**: 地铁站、公交车上、部分出租车。
    -   **公共场所**: 机场、火车站、图书馆、医院、旅游景点。
    -   **商业区域**: 绝大多数的酒店、咖啡馆、餐厅和购物中心。

## 国际漫游

-   **成本**: 通常是**最昂贵**的选择，费用可能非常高。
-   **建议**: 仅作为紧急备用方案。出发前请咨询您的运营商具体的漫游费用。

## 最终建议

-   **个人便捷首选**: **eSIM**。提前购买，落地激活，无忧上网。
-   **团体/多设备首选**: **便携Wi-Fi**。共享网络，性价比高。
-   **传统可靠之选**: **物理SIM卡**。兼容性强，套餐选择多。
