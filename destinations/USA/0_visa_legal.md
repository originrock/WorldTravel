# 美国签证与法律须知

## 签证 (Visa)

前往美国进行旅游活动，主要有两种常见的签证类型，具体选择取决于您的国籍和旅行目的。

1.  **旅游签证 (B-2 Visa)**:
    -   适用于不符合免签计划条件的国家公民，或希望在美国停留超过90天（尽管本次行程是60天，但如果后续有更长停留计划，或出于其他原因，B2是选项）。
    -   **申请流程**:
        1.  填写DS-160非移民签证电子申请表。
        2.  支付签证申请费。
        3.  预约使领馆面谈。
        4.  前往使领馆进行面谈，可能需要提供旅行计划、财力证明、回国意愿等材料。
    -   **有效期**: 通常为10年多次入境，每次停留期由入境时海关官员决定，最长可达6个月。

2.  **免签证计划 (Visa Waiver Program - VWP) & ESTA**:
    -   适用于39个国家和地区的公民（包括大部分欧洲国家、日本、韩国、澳大利亚、新西兰等）。这些国家的公民如果满足特定条件，可以在没有签证的情况下前往美国旅游或商务，最长停留90天。
    -   **ESTA (Electronic System for Travel Authorization)**: 即使符合免签条件，也必须在前往美国前在线申请并获得ESTA批准。
    -   **申请流程**:
        1.  访问美国海关和边境保护局官方网站 (esta.cbp.dhs.gov) 申请。
        2.  填写个人信息、旅行信息、安全问题等。
        3.  支付$21美元费用。
        4.  通常在几分钟内即可获得批准，但建议至少在出发前72小时申请。
    -   **有效期**: ESTA批准有效期通常为2年，或直到您的护照过期（以先到者为准）。在有效期内，可以多次入境美国，每次停留不超过90天。

**重要提示**: 无论何种签证，入境时美国海关及边境保护局 (CBP) 官员拥有最终决定权，决定您是否可以入境以及停留时间。请准备好您的旅行计划、住宿预订、返程机票等以备查验。

## 法律须知

1.  **酒驾 (DUI/DWI)**:
    -   在美国，酒驾是非常严重的违法行为，将面临巨额罚款、吊销驾照、甚至监禁。各州对血液酒精含量 (BAC) 的法定上限通常为0.08%。
    -   严格遵守“不酒驾”原则。
2.  **毒品**:
    -   大麻在某些州（如加州、科罗拉多州、华盛顿州）已经合法化用于娱乐用途，但联邦法律仍然禁止。在联邦土地（如国家公园）和机场，大麻仍然是非法的。
    -   其他毒品在全美范围内均属非法。
3.  **吸烟**:
    -   多数公共室内场所（包括餐厅、酒吧）禁止吸烟。许多酒店为无烟酒店。
    -   部分州和城市对电子烟也有严格的规定。
4.  **枪支**:
    -   美国枪支法律各州差异巨大且复杂。作为游客，请避免接触枪支，更不要尝试购买或携带。
5.  **紧急情况**:
    -   **报警/急救/消防**: 拨打 **911** (全美通用免费电话)。
    -   **使领馆**: 遇到领事保护问题，可联系中国驻美国使领馆。
6.  **购物税 (Sales Tax)**:
    -   商品标价通常不含销售税，税率各州不同，购物时会在结账时加上。
    -   部分州（如俄勒冈、蒙大拿、特拉华、新罕布什尔）免销售税。
7.  **垃圾分类**:
    -   不同州和城市可能有不同的垃圾分类规定，请留意酒店或民宿的指引。

**建议**: 出行前查阅最新、最准确的签证和入境信息，并熟悉目的地州的具体法律法规。随身携带护照和签证复印件，以备不时之需。
