# 韩国30天旅行预算估算

本文件提供一个为期30天的韩国旅行大致预算，基于**中档旅行者**的消费水平。所有金额均为估算，并以韩元(KRW)为单位，实际花费会因个人消费习惯、季节和预订时间而异。

**汇率参考**: 1美元 ≈ 1,300韩元 (请以出行时最新汇率计算)

---

## 一、每日预算估算 ( per person / per day )

这是一个平均每日花费的参考，可根据不同城市的消费水平进行微调。

| 类别 | 预算 (KRW) | 备注 |
| :--- | :--- | :--- |
| **住宿 (Accommodation)** | ₩80,000 | - 基于经济型酒店或精品民宿的平均价格。<br>- 青旅床位约 ₩30,000-₩40,000；高档酒店则在 ₩150,000以上。 |
| **餐饮 (Food)** | ₩60,000 | - 早餐: ₩10,000 (便利店/咖啡馆)<br>- 午餐: ₩15,000 (当地餐厅简餐)<br>- 晚餐: ₩25,000 (烤肉/特色料理)<br>- 零食/饮料: ₩10,000 |
| **市内交通 (Local Transport)** | ₩10,000 | - 基于每日乘坐3-4次地铁或公交车的费用。 |
| **景点/活动 (Activities)** | ₩20,000 | - 这是平均值。部分宫殿门票约 ₩3,000，而DMZ一日游等特殊项目费用会更高。 |
| **每日总计 (Daily Total)** | **₩170,000** | **约 130 美元/天** |

## 二、30天总预算估算 ( per person )

### 1. 每日花费部分

-   **每日总计**: ₩170,000
-   **天数**: 30天
-   **小计**: ₩170,000/天 * 30天 = **₩5,100,000**

### 2. 主要城际交通花费 (一次性)

-   **首尔 → 釜山 (KTX高铁)**: 约 **₩60,000** (经济座)
-   **釜山 → 济州岛 (飞机)**: 约 **₩50,000** (廉价航空，含行李)
-   **济州岛 → 首尔 (飞机)**: 约 **₩50,000** (廉价航空，含行李)
-   **小计**: **₩160,000**

### 3. 其他一次性花费

-   **SIM卡/eSIM (30天)**: 约 **₩40,000**
-   **T-Money卡成本**: 约 **₩5,000**
-   **济州岛租车 (3天，含保险)**: 约 ₩150,000 / 2人 = **₩75,000** (按两人分摊)
-   **小计**: **₩120,000**

**最终估算**: 一个人30天的韩国中档旅行，总预算大约在 **550万至600万韩元** 之间 (约 **4,200 - 4,600 美元**)。

## 预算节省技巧

1.  **住宿**:
    -   选择弘大、西面等区域的精品旅馆或青旅。
    -   考虑使用Airbnb。
2.  **餐饮**:
    -   多利用便利店的简餐和饭团。
    -   在大学周边用餐，价格通常更实惠。
    -   在传统市场解决午餐。
3.  **交通**:
    -   充分利用T-Money的换乘优惠。
    -   天气好时，多采用步行探索城市。
4.  **景点**:
    -   许多宫殿在每月最后一个周三是“文化日”，可以免费或优惠入场。
    -   利用首尔转转卡(Discover Seoul Pass)等通票，如果计划参观大量收费景点。
